## Clonar el repo

Abrir en la carpeta donde quieres clonar el repo el bash de git y hacer esto:

git clone https://github.com/964jorge/Practica-3-bd-avanzadas

Luego para trabajar cierra el bash y abrelo dentro de la carpeta.

## Pullear

git pull origin main

## Pushear

git add .

git commit -m "Descripción de los cambios"

git push origin main


## Buenas prácticas

- Hacer `pull` antes de empezar a trabajar.
- Escribir mensajes de commit claros.
- Realizar commits pequeños y frecuentes.
- Usar `.gitignore` para evitar subir archivos innecesarios.
